# BlazorWebAssemblyPublish

## Running the example using http-server npm package
### Download and install node.js
https://nodejs.org/en/
### Install http-server package globally
```
npm i -g http-server
```
### Host the sample
```
cd BlazorWebComponentsPub
http-server
```
### Open the sample from a web browser
http://127.0.0.1:8080
